# Signal Analysis Lab (SAL)

Strategic Communication Intelligence - Decode any conversation for power dynamics, behavioral patterns, and optimal response strategies.

**Live App:** [signal-analysis-lab.vercel.app](https://signal-analysis-lab.vercel.app)

## Features

- **Conversation Parser** - Supports WhatsApp exports, iMessage, Telegram, or any "Speaker: message" format
- **Win Probability** - AI-calculated likelihood of achieving your stated objective
- **Power Dynamics** - Visual balance of leverage between parties
- **Behavioral Profiling** - Communication style, decision speed, attachment patterns, hidden priorities
- **Your Communication DNA** - Discover your archetype, strengths, and watch points
- **Critical Moments** - Timeline of turning points with leverage impact scores
- **Signal Detection** - Green lights and red flags with weighted importance
- **Optimal Timing Engine** - Best days, hours, and response delays for engagement
- **Strategic Message Composer** - Multiple response variants with annotations explaining the psychology behind each phrase
- **Move Tree** - Predicted responses and your optimal counter-moves

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons

## Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Privacy

All processing happens locally in your browser. Your conversations are never stored or transmitted to any server.
